# Quick Start Guide - IT Admin Portal

## Fastest Way to Launch

1. **Double-click** `Launch-ITAdminPortal.bat` in the project root
   - OR
2. **Run** `.\Launch-ITAdminPortal.ps1` in PowerShell

## First Time Setup

1. Ensure .NET 6.0 SDK is installed
2. Run the launcher - it will build automatically on first run
3. The portal will open and display all available scripts

## Create Desktop Shortcut

Run this once to create a desktop shortcut:
```powershell
.\Create-DesktopShortcut.ps1
```

Then just double-click the desktop icon anytime!

## Project Location

The C# WPF project is located at:
```
.\ITAdminPortal\ITAdminPortal.csproj
```

## Build Output

After building, the executable is at:
```
.\ITAdminPortal\bin\Debug\net8.0-windows\ITAdminPortal.exe
```
or
```
.\ITAdminPortal\bin\Release\net8.0-windows\ITAdminPortal.exe
```
